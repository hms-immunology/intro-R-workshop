# Setup Instructions

## 1. Install R

1. Go to [CRAN](https://cran.rstudio.com/)
2. Follow the instructions for your operating system:

   **For Apple (macOS) computers:**
   - Click "Download R for macOS"
   - Under "Latest release:", choose the appropriate version:
     - For Apple Silicon (M1/M2 processors): Select the arm64 version
     - For Intel processors: Select the x86_64 version
   - Not sure which chip you have? Click [here](https://support.apple.com/en-us/HT211814) to find out
   - Double click the downloaded file and follow the installation instructions

   **For Windows computers:**
   - Click "Download R for Windows"
   - Click "base"
   - Click the download link for the latest version
   - Open the downloaded file and follow the installation instructions

## 2. Install RStudio

1. Go to [RStudio Downloads](https://posit.co/download/rstudio-desktop/)
2. Download RStudio Desktop (Free version)
   - Skip step 1 on the website (you've already installed R)
   - Under step 2, the website should automatically detect your operating system and offer the correct download
3. Open the downloaded file and follow the installation instructions
